Poems_generator_Keras
========
 
- 唐诗，古诗，五言绝句自动生成，基于Keras，LSTM-RNN。       
- 附带训练好的模型文件，可直接上手使用。         
- 功能：藏头诗，随机写诗，给定第一句诗/字进行作诗
